Школьный проект "Космическое путешествие" для младшей сестры.

Включает в себя краткий обзор с интересными фактами о 4 планетах солнечной системы, а также в отдельной вкладке "Земплеподобные планеты" представлены планеты похожие на землю в других звездных системах.
